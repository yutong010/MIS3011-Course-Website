Advanced AI for Business: From Theories to Applications

MIS3011 | CUHKSZ | Fall 2025

Course Overview

Mastering AI Technologies for Business Innovation

English

This course builds comprehensive AI understanding for business students, covering classic to cutting-edge technologies with consistent focus on practical application value. Students master core competencies to solve business problems using AI.

Key topics: (1) machine learning methods; (2) neural networks and deep learning applications; (3) Generative AI technologies; (4) AI application development.

Hands-on assignments and group projects simulate real business scenarios, developing technical and project management skills.

Learning Outcomes

This course focuses on three core skill pillars that prepare you for AI applications in business:

Skill Pillar 1
Predicting the Future with Data
  • Master Logistic Regression for business decisions
  • Deep dive into Neural Networks and Deep Learning
  • Build a customer churn prediction model for a bank
  • Act as a data scientist to predict personal loan offer acceptance
Skill Pillar 2
Teaching Computers to See
  • Learn about powerful Convolutional Neural Networks (CNNs)
  • Master the efficient technique of Transfer Learning
  • Build a "Hot Dog/Not Hot Dog" image classifier from scratch
  • Develop a touchless hand navigation control system using MediaPipe
Skill Pillar 3
Mastering the Language of AI
  • Learn essential techniques for Natural Language Processing
  • Use Hugging Face library for sentiment analysis
  • Look inside the black box of LLMs
  • Explore Transformers to GenAI Project Lifecycle
